# 截图工具

轻量级 Windows 截图工具，固定区域一键截图自动存入 + 手动框选自动存入，以悬浮球形态常驻桌面。

---

## 安装

双击 `ScreenshotTool.exe`，无需安装。

> **杀毒软件误报**：工具使用了键盘监听和屏幕捕获，可能被误判为可疑程序。
> 在杀毒软件里把 `ScreenshotTool.exe` 加入白名单即可正常使用。

---

## 使用方法

### 悬浮球操作

| 操作 | 效果 |
|---|---|
| **单击** | 展开 / 收起设置面板 |
| **拖动** | 把悬浮球移到屏幕任意位置 |
| **右键** | 菜单：设置 / 关于 / 退出 |

### 截图快捷键

| 默认快捷键 | 功能 |
|---|---|
| `Ctrl+Alt+S` | 固定区域截图（按预设坐标） |
| `Ctrl+Alt+D` | 手动框选截图 |

截图自动保存到设置的目录，文件名格式：`screenshot_20260520_143022.png`

---

## 配置预设

1. 单击悬浮球打开设置面板
2. 点 **📐 标定区域**，窗口隐藏后用鼠标拖框选定截图区域
3. 坐标自动回填，填写快捷键（如 `ctrl+alt+1`）
4. 点 **💾 保存配置**

可以添加多个预设，每个预设有独立的区域和快捷键。

---

## 常见问题

**Q：快捷键没有反应？**
右键 `ScreenshotTool.exe` → 属性 → 兼容性 → 勾选"以管理员身份运行"

**Q：截图坐标偏移？**
检查 Windows 显示设置里的缩放比例。工具已处理常见缩放（100% / 125% / 150% / 200%），
若仍偏移请使用"标定区域"重新设定坐标。

**Q：杀毒软件一直拦截？**
将 exe 文件路径加入杀毒软件的白名单 / 排除列表。

---

## 截图保存位置

默认：`C:\Users\你的用户名\Pictures\Screenshots\`

可在设置面板底部修改。
